# `edda`

</div>

> NOTE: Currently in early development. Study project.

Edda is a lightweight, embedded storage engine. It combines key-value storage, vector search, and LLM response caching in a single binary. Named after the [Poetic Edda](https://en.wikipedia.org/wiki/Poetic_Edda).
